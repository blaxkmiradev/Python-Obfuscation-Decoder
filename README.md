# Python Obfuscation Decoder

A simple Python tool to decode multi-layered obfuscated `.py` files (base64 + zlib).

## Usage

```bash
python decoder.py
```

The script will prompt you to enter the file path to decode. Once decoded, it saves the output as `<filename>_decoded.py`.

## Requirements

- Python 3.x
- Standard libraries: `base64`, `zlib`, `re`, `os`
